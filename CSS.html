About CSS!

Learning web development basic technology needs to know three sowrds: HTML, CSS and JavaScript language.

CSS:Css(English Cascading Style Sheets) is a computer language that is used to represent HTML (a standard generic markup language application) or XML (a subset of Standard Generalized Markup Language). CSS can not only modify web pages statically, but also dynamically formatting the elements of a web page in a variety of scripting languages.
CSS can precisely control pixel layout in web pages, support almost all font and font styles, and have the ability to edit webpage objects and model styles.

Language characteristics

Rich style definition
 CSS provides a rich document style appearance, and the ability to set the text and background attributes; allowed to create borders for any element, and the distance between elements and other elements of the frame, and the distance between the frame and the content of the element element; allowed to change the text size and writing, modification and other page effect.
Easy to use and modify
 CSS can define styles in the style attribute of HTML elements, or it can be defined in the header part of HTML documents, or it can be declared in a specialized CSS file for HTML page reference. In a word, the CSS stylesheet can store all the style declarations in a unified way and manage them in a unified way.
In addition, you can classify the same style elements, use the same style to define, or apply a style to all the HTML tags of the same name, or you can assign a CSS style to a page element. If the style is to be modified, we only need to find the corresponding style declaration in the style list to modify it.
Multi page application
 The CSS stylesheet can be stored in a single CSS file, so that we can use the same CSS stylesheet on multiple pages. The CSS stylesheet does not belong to any page file in theory, and can be referenced in any page file. This can achieve the unity of multiple page styles.
Cascade
 In a simple way, stack is to set the same style many times for an element, which will use the attribute value of the last set. For example, multiple pages in a site use the same set of CSS stylesheets, while some elements in some pages want to use other styles. You can define a stylesheet to apply to the page individually for these styles. These later defined styles will rewrite the previous style settings, and what you see in the browser will be the most later set of style effects.
Page compression
 In websites that use HTML to define page effects, a large number of duplicate tables and font elements are needed to form various types of text styles. The consequence of this is that a large number of HTML tags will be generated, which will increase the size of page files. The style declaration can be reduced to the CSS stylesheet alone, which can greatly reduce the volume of the page, so that the time to load the page will be greatly reduced. In addition, the reuse of the larger CSS stylesheets reduces the volume of the page and reduces the time to download.

Working principle

CSS is a language that defines style structures, such as fonts, colors, positions, and so on, which is used to describe the format of information formatting and reality on a web page. The CSS style can be stored directly on a HTML web page or a separate style single file. In either way, the style sheet contains the rules that apply the style to the specified type of element. In external use, the style single rules are placed in an external style single document with the file extension name _css.
Style rules are formatted instructions that can be applied to elements in a web page, such as text paragraphs or links. Style rules are made up of one or more style properties and their values. The internal style sheet is placed directly on the web page, and the external style sheet is guaranteed to exist in a separate document, and the web page links the external style sheet through a special label.

Technical application

Add a hyperlink in the HTML file and introduce an external CSS document. This method is the most convenient way to manage the web style of the whole web site, which separates the text content from the layout design. As long as we define the style of the web page in a CSS document (extension CSS), then add a hyperlink to the document in the webpage, then the web page will be displayed according to the well-defined style in the CSS document.
