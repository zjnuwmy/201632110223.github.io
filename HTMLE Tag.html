Characteristic

1.Key words encircled by sharp brackets, such as <html>
2.Usually in pairs, such as <div> and </div>
3.The first label in the label pair is the start tag, and the second label is the end tag.
4.The start and end tags are also called open and closed tags.
5.There are separate presentation tags, such as: <img src= "Baidu Encyclopedia.Jpg" etc..
6.A general pair of labels that appear in the middle of the two labels. The tags that are presented separately are assigned in the label properties. Such as <h1> </h1> and <input type= title "text" value= "button".
7.The contents of web pages need to be displayed in <html> tags. The information of title, character format, language, compatibility, keywords and description is displayed in <head> tags, and the content displayed on Web pages should be nested in <body> tags. In some cases, the non - standard writing code can be shown normally, but as a professional accomplishment, formal writing habits should be developed.
