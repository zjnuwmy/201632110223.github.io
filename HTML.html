About HTML!

Learning web development basic technology needs to know three sowrds: HTML, CSS and JavaScript language.

HTML:HTML is the abbreviation of hypertext markup language,it is an application under standard General Markup Language( An international standard language that defines the structure of electronic documents and describes their contents). Hypertext refers to non text elements that contain pictures, links, and even music, programs, and so on.
English name: HyperText Markup Language Reading methods: Web browser
Type: Markup Language Abbreviation: HTML or HTM (both file extensions)

Origin

The essence of web pages is hypertext markup language, which can create powerful web pages by using other Web technologies (such as scripting language, public gateway interface, components and so on). Therefore, hypertext markup language is the foundation of the world wide web (Web) programming, that is to say, the world wide web is based on hypertext. Hypertext markup language is called hypertext markup language because the text contains the so-called "hyperlink" point.

Definiton

Hypertext markup language is an application of standard general markup language. It is also a standard. It marks all parts of the web page which is displayed by mark symbol. Web page file itself is a text file, by adding markup in the text file, you can tell the browser how to display the contents (such as: how the text processing, how to arrange the picture, how to display the picture, etc.). The browser reads the web page in sequence, and then explains and displays the contents of the tag according to the token.
For different browsers, there may be different interpretations of the same token, which may have different display effects.
